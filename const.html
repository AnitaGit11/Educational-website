Introduction to C

What is C?

  • C is a general-purpose programmiglanguage developed at AT & T Bell Laboratories of USA in 1972.
  • It was designed and written by a man named Dennis Ritchie.
  • In the late seventies C began to replace the more Languages of that time like PL/I, ALGOL, etc.
  • It is a high level language.
  • It is a successor to B language.
  • C is strongly associated with UNIX, as it was developed to write the UNIX operating system.

Why Learn C?

  • It is one of the most popular programming language in the world
  • If you C, you will have no problem learning othr popular programming languages such as Java, Python, C++, C#, etc, as the syntax is similar
  • C is very fast, compared to other programmig languages, like Java and Python
  • C is very versatile; it can be used in both applications and technologies
